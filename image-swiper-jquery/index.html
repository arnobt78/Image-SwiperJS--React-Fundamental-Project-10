Shopify Style Image Swiper using jQuery and Swiper.js

Learn: This project uses jQuery (DOM manipulation) and SwiperJS (touch-enabled slider). Each slide has its own .text-overlay inside the slide, so the text swipes with the image. The nav bar (arrows + dots) is built in script.js via Swiper's on.init callback: jQuery moves the prev/next buttons into the pagination container so they appear centered together.
